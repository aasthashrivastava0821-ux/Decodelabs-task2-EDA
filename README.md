# Decodelabs Task 2 - E-Commerce EDA & Dashboard

## Overview
This project focuses on Exploratory Data Analysis (EDA) of an E-Commerce dataset and the creation of an interactive Power BI dashboard to generate business insights.

## Objectives
- Analyze sales performance
- Identify customer purchasing trends
- Discover top-performing products and categories
- Create visual reports and dashboards

## Tools Used
- Python
- Pandas
- Power BI

## Analysis Performed
- Data cleaning and preprocessing
- Sales trend analysis
- Customer behavior analysis
- Product performance analysis
- Data visualization

## Dashboard Features
- Total Sales
- Total Orders
- Top Products
- Sales by Category
- Customer Insights

## Files Included
- eda.py
- Power BI Dashboard (.pbix)
- Dashboard Screenshots
- README.md

## Outcome
Generated actionable business insights from the E-Commerce dataset and presented them through an interactive dashboard.
